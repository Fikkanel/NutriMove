UAS Pemrograman Mobile
Mata Kuliah: Pemrograman Mobile (Flutter)
🎯 Deskripsi
Mahasiswa diminta untuk mengembangkan aplikasi mobile berbasis Flutter secara end-to-end yang mengimplementasikan fitur utama pengembangan aplikasi modern, meliputi CRUD, database, autentikasi, dan animasi.
Aplikasi bersifat bebas (open theme), namun wajib memenuhi seluruh kriteria teknis di bawah.
⚙️ REQUIREMENT UTAMA (WAJIB)
1. 🔄 CRUD (Create, Read, Update, Delete)
Aplikasi harus memiliki fitur:
Create → Menambahkan data
Read → Menampilkan data (list/detail)
Update → Mengubah data
Delete → Menghapus data
📌 Contoh:
Todo App
Catatan keuangan
Kuis (dengan manajemen soal)
Aplikasi booking
2. 🎨 Animation
Hero Animation
Progress Indicator Animation
Lottie / Rive Animation
Confetti Animation
Animated List
FadeTransition
ScaleTransition
🧱 STRUKTUR MINIMAL APLIKASI
Halaman Utama (Dashboard/List Data)
Halaman Tambah/Edit Data
Halaman Detail
Animasi di minimal 2 bagian
📊 MATRKS PENILAIAN
Table
🚀 NILAI TAMBAHAN (BONUS)
🟢 Level 1 – Enhancement Dasar (+5% per fitur)
Fitur yang meningkatkan UX tapi masih relatif mudah:
Dark Mode / Theme Switching
Form Validation (email, password, dll)
Empty State UI (tidak hanya blank screen)
Loading Indicator yang proper
Error Handling (Snackbar/Dialog)
Pull to Refresh
Search & Filter sederhana
Sorting data
🟡 Level 2 – Intermediate (+10% per fitur)
Butuh pemahaman lebih dalam dan integrasi fitur:
State Management (Provider / Riverpod / BLoC)
Pagination / Infinite Scroll
Local Cache (Hive/SQLite) + sync sederhana
Multi-role user (admin vs user)
Upload image/file ke Firebase Storage
Custom reusable widgets/component system
Form kompleks (multi-step form)
🔵 Level 3 – Advanced (+15% per fitur)
Mulai masuk ke level “capstone serius”:
REST API integration (backend eksternal)
Push Notification (Firebase Cloud Messaging)
Offline-first app (data tetap jalan tanpa internet)
Advanced animation (Rive / complex transition)
Data visualization (chart/graph)
Security rules Firebase (bukan default open)
🔴 Level 4 – Expert (+20% per fitur)
Level yang biasanya jarang dicapai (nilai A++ kalau rapi):
Clean Architecture (MVVM / Repository Pattern)
Dependency Injection
Automated Testing (unit / widget test)
AI integration (misalnya rekomendasi, chatbot sederhana)
Gunakan tema olahraga
Firebase
