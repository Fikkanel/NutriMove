# Changelog — NutriMove

### 🎉 Initial Release

#### Added
- **Core Architecture**: Clean Architecture dengan Flutter 3.41.1
- **Design System**: Dark theme premium, Outfit/Inter typography, 30+ color tokens

- **NutriBot**: AI chatbot dengan suggestion chips

- **Shared Widgets**: NutriMoveButton, TextField, AnimatedCard, BottomNavBar, LoadingOverlay

- **State Management**: 7 global providers via Provider package
- **Testing**: Unit tests untuk Fuzzy AHP, TOPSIS, Auth, Gamification
- **Documentation**: README, Architecture, Database Schema, Environment, Deployment, SRS, FRS
